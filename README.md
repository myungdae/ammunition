# Agentic AI Platform Demo

Palantir 스타일의 Agentic AI 데모 플랫폼입니다. 제안서 및 프레젠테이션용으로 제작되었습니다.

## 🌟 Features

- **실시간 대시보드**: 에이전트 활동 및 시스템 메트릭 실시간 모니터링
- **다크 테마**: Palantir 스타일의 미래지향적 UI/UX
- **인터랙티브 차트**: Canvas 기반의 실시간 데이터 시각화
- **에이전트 관리**: AI 에이전트 상태 및 작업 모니터링
- **시스템 헬스**: CPU, 메모리, 네트워크, 스토리지 모니터링
- **라이브 피드**: 실시간 시스템 이벤트 로그
